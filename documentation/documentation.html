Template Features:

  1. Endless possibilites – With the advanced styling customization features, you can create endless possibilities by changing navigation styles, backgrounds for each section with colors, images or patterns.
  2. Ultra Responsive – Newave is 100% responsive, each and every element including the awesome slider are fully responsive.
  3. Parallax Sections – Newave is built using Parallax javascript. This enables to make your site more appealing and creative. It's also very easy to use.
  4. Custom Backgrounds – Newave allows you to set a custom background for each section. A super-quick way to get creative with your site. Parallax sections makes your page more beautiful.
  5. 5 Color Skins – Newave includes 5 color skin verisons. You can create new one very easy. They have separate css files
  6. 4 Header versions – Newave includes 4 header verisons. You can customize each of them very easy. They have separate css files
  7. 5 Home Section versions – Home section with background pattern and text slider, Home section with fullscreen slider and animated captions, Amazing Home section with parallax slider and animated captions, Home section with parallax image and text slider, Home Section with video background
  8. Isotope Portfolio - We create a fullwidth 4 colums portfolio, that never breaks rows. Always display properly. We guarantee this!
  9. Project Ajax Normal Expander - Includes a slider and HTML content. It also supportss media like youtube, vimeo, etc.
  10. Project Ajax Expander - Include a fullscreen slider with slide captions. Very nice for photography
  11. Working Contact Form - The elements of the contact form is perfectly styled and very easy to customize it.
  12. Font Awesome Integration - We include over 240+ Icon font integration. You can use these icons via shortcodes in any page. You can customize the size,color,type of the icon.
  13. Browser Compatability - This template supports all the major browsers including IE9+, Chrome, Safari, Firefox.
  14. Clean Code - The code is well written using the best resources available and also all the files are well commented to make your work much easier.
  15. Documentation - We include the detailed documentation on how to use the theme and it's features. Plus you can always submit a ticket at our heldeks if you have any queries.

Getting Started

After unzip template Files, you will found www folder that include all the temeplate files.


You can view this template in any web browser from your desktop computer. Because some js files are external, you need an internet connection to display correctly the template.


Start your FTP client (I persoanlly use Total Commander) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.


Locate the domain folder you wish to upload your Theme files to on your server. You are going to want to click and drag every file into your FTP client.
